# Stars-Animation

A simple, lightweight starfield animation built with HTML, CSS and JavaScript.

![Preview](./Screenshot%202024-07-26%20002226.png)

## Features

- Smooth animated starfield background
- Tiny, dependency-free code (vanilla HTML/CSS/JS)
- Easy to embed or customize for websites and demos

## Files

- `index.html` — minimal HTML that loads the animation
- `style.css` — small stylesheet for layout
- `script.js` — JavaScript that creates and animates the stars
- `Screenshot 2024-07-26 002226.png` — preview image used above

## Getting started

1. Clone or download the repository:

   git clone https://github.com/BinaryVortex/Stars-Animation.git

2. Open `index.html` in your browser.

Alternatively, run a simple local web server (recommended) and visit http://localhost:8000:

- Python 3: `python -m http.server 8000`

## Customization

- Edit `script.js` to change star count, speed, size and behavior.
- Tweak colors and layout in `style.css`.

## Usage

Drop the contents into any web project or embed the relevant parts in an existing page. The animation is lightweight and suitable for headers, backgrounds, and demos.
